## How to Compile and Run

### Compile a single Java file:
```bash
javac Assignment_X/QY.java
```

### Run the compiled program:
```bash
java -cp Assignment_X QY
```

### Example:
```bash
# Compile
javac Assignment_5/Q1.java

# Run
java -cp Assignment_5 Q1
```
